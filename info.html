01  這次做了什麼

把 V0 的基礎架構一件一件立好，每件對應一條（或一組）ADR、每件是一個合理 commit，最後切成 7 個堆疊式（stacked）PR 發回 main，方便你逐一 review。

地基 only。 本次只安裝依賴並產出資料夾／檔案骨架，確保 build · dev · test · lint 全通。地圖實際渲染、手繪假禁航多邊形、點選 advisory 資訊卡、免責常駐入口／首次確認都還沒做，會在後續 feature PR 一件一件接上（見最後一節）。

07  怎麼 review / 跑起來

Review 順序：從 #1 開始由下往上。要改哪個 PR 直接在該 PR 留言，我會修正並把上層 rebase 跟上（堆疊維護由我負責）。

本機把 app 跑起來（完整 app 在最上層 branch）：

git checkout test/vitest-setup
pnpm install
pnpm dev        # Vite dev server
pnpm build      # tsc --noEmit && vite build
pnpm test       # vitest run
pnpm lint       # biome check .

main 目前等同 origin/main（scaffold 之前），合併完 PR 後 git checkout main && git pull 即得全部。
